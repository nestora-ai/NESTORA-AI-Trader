# NESTORA AI Trader v3.0 — Phase 1

المرحلة الأولى من إعادة بناء المنصة بصورة احترافية.

## ما تم إنجازه
- واجهة حديثة متعددة الصفحات
- لوحة تحكم احترافية
- رسوم بيانية تفاعلية محلية
- صفحة أسواق
- مساعد NESTORA AI محلي بقواعد تحليلية
- حاسبة مخاطر
- سجل صفقات مع حفظ محلي وتصدير CSV
- صفحات الحسابات والإعدادات
- تصميم متجاوب مع الهاتف والكمبيوتر

## ما لم يتم ربطه بعد
- أسعار السوق الحية
- الذكاء الاصطناعي السحابي
- cTrader OAuth
- Exness MT5 backend
- تسجيل الدخول وقاعدة البيانات

> هذه النسخة تعليمية ولا تنفذ صفقات تلقائيًا ولا تضمن الأرباح.
